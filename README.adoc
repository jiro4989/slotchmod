= slotchmod
:sectnums:
:toc: left

`slotchmod` changes file permission with a slot.
This is a useful tool when you want to mess up file access permissions.

image:./docs/demo.gif[]

== Usage

[source,bash]
----
slotchmod [files...]
----

== Installation

[source,bash]
----
go get github.com/jiro4989/slotchmod
# or
go install github.com/jiro4989/slotchmod@master
----
